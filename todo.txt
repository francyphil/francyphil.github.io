- Card responsive
    Problema: Le card hanno width: calc((100% - 40px) / 3) fisso
    Proposta: Aggiungere media queries:

    @media (max-width: 768px) {
    .category-card { width: calc((100% - 20px) / 2); } /* 2 per riga */
    }
    @media (max-width: 480px) {
    .category-card { width: 100%; } /* 1 per riga */
    }
